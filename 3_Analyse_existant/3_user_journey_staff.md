# User journey staff

## 1. Persona

| Profil | |
| --- | --- |
| **Nom** | Thomas |
| **Âge** | 24 ans |
| **Rôle** | Serveur |
| **Descriptif de poste** | Étudiant qui travaille en restauration pour financer ses études. Tourne parfois sur plusieurs établissements, doit s’adapter rapidement à de nouveaux outils sans formation longue. Travaille sous pression en heure de pointe. |

| **Objectifs** | **Besoins** |
| --- | --- |
| - gérer efficacement les commandes - éviter les erreurs de prise de commande - gagner du temps en service | - interface claire et lisible - accès rapide aux commandes - outil simple à utiliser |

| **Défis** | **Comportement digital** |
| --- | --- |
| - erreurs de commande possibles - gestion du stress en heure de points - communication avec la cuisine | - à l’aise avec les outils simples - préfère des interfaces rapides - peu de temps pour apprendre |

## 2. Citation personnelle

> *Je veux voir rapidement les commandes sans me compliquer la vie.*

## 3. Analyse

Thomas travaille dans l’urgence, zéro tolérance pour une interface chargée. Le dashboard staff doit afficher uniquement les informations importantes, avec des actions accessibles en un seul clique.

## 4. User journey

|  | **Étape 1** | **Étape 2** | **Étape 3** | **Étape 4** | **Étape 5** |
| --- | --- | --- | --- | --- | --- |
| **Action de l'utilisateur** | Se connecter à l’interface staff | Consulter les commandes | Mettre à jour  le statut | Modifier le plan de table | Fermer le service |
| **Objectif** | Accéder rapidement aux commandes | Comprendre les commandes en cours | Suivre l’avancement du service | Adapter la salle en temps réel | Clôturer le service proprement |
| **Émotion** | 😐Neutre | 😟Stressé | 🙂Soulagé | 😊En contrôle | 😊Satisfait |
| **Point de friction** | Connexion lente ou avec trop d'étapes | Interface difficile à lire en rush | Modification nécessitant plusieurs actions | Fusion de tables peu intuitives | Oublie de fermeture |
| **Opportunité et solution** | Connexion rapide sans étapes inutiles | Dashboard épuré avec informations essentielles | Boutons rapides pour changer le statut en un tap | Fusion simple avec regroupement automatique des commandes | Configuration de fermeture avec rappel des commandes en cours |
